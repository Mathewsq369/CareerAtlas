# CareerAtlas
An AI-powered personality-based career guidance and learning pathway recommender system for students




1.1 Project Architecture
```
edupersona/
├── backend/                 # Django backend
│   ├── core/               # Main settings
│   ├── users/              # User management
│   ├── assessments/        # Personality tests
│   ├── recommendations/    # Career/subject recommendations
│   ├── ai_coach/          # AI coaching system
│   └── analytics/         # Data analysis
├── frontend/              # React/Vue (optional - can use Django templates)
├── data/                  # MBTI and career data
└── docs/                  # Documentation
```
